# \# D20 Roll Demo

# 

# A Unity prototype inspired by the dice roll mechanic from Baldur's Gate 3.

# 

# \## About

# 

# This project is a technical test assignment for a Unity Developer position.

# 

# The goal is to recreate the core gameplay experience of rolling a D20 dice, including:

# 

# \- D20 dice roll mechanic

# \- Random roll results (1–20)

# \- Roll modifiers

# \- Result presentation

# \- Critical Success and Critical Failure

# \- Smooth UI animations and visual feedback

# 

# The project focuses on clean architecture, readable C# code, and a polished user experience.

# 

# \## Built With

# 

# \- Unity

# \- C#

# \- TextMeshPro

# \- Unity UI

#
